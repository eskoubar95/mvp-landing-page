# Assets - Reference Material

Denne mappe indeholder reference materiale fra Rentumo.com til brug ved design og implementering af MVP landing page.

## Struktur

```
assets/
├── reference-images/     # Screenshots og reference billeder fra Rentumo
└── README.md            # Denne fil
```

## Reference Billeder

### Screenshots
- **Full Page Screenshot:** `.playwright-mcp/page-*.png` - Fuld side screenshot fra Rentumo.com hjemmeside

### Brug af Reference Materiale

**Vigtigt:** 
- Billeder og design elementer fra Rentumo bruges kun som **inspiration og reference**
- **IKKE** direkte kopiering af billeder eller assets
- Opret egne illustrationer og billeder baseret på stil og æstetik
- Respekter copyright - brug kun som visuel reference

## Design Inspiration

Reference materialet bruges til at forstå:
- Farvepalette og farvevalg
- Typografi og font styling
- Layout patterns og spacing
- Component designs
- Visual hierarchy
- Illustration stil
